# 成片验收清单

AI 在输出视频成片前必须逐项自检（与开发自检清单互补，此清单聚焦成品质量）。

## 画面与布局（成品视角）
- [ ] 所有公式、矩阵、文字未超出画面边界。
- [ ] 无残留元素（如动画切换后旧等号未消失）。
- [ ] 间距正常，无异常拥挤或重叠。
- [ ] 等号、矩阵列、上下标对齐正确。
- [ ] 字体一致，无乱码或混用。
- [ ] 视频分辨率（如 1080p）下所有内容清晰可读。
- [ ] 默认布局为单栏按区域要求居中。
- [ ] 标题区位于 Y = 3.6（如无标题则跳过）
- [ ] 副标题区位于 Y = 3.1 或紧贴标题下方
- [ ] 主内容区从顶部开始排列（首个元素 Y = 3.2）
- [ ] 无主内容时，标题/副标题整体垂直居中
- [ ] 分栏时各栏顶部对齐，整体水平居中
- [ ] 图形区位于右侧，与主内容区顶部对齐（两栏/三栏模式时）
- [ ] 图形内容清晰可辨，无变形或重叠

## 音频与字幕
- [ ] 视频有声音（非静音）。
- [ ] 音画同步（讲解时机与画面出现一致）。
- [ ] 字幕准确，无错别字、漏句、不同步。
- [ ] 字幕不遮挡关键公式/矩阵元素。
- [ ] 数学符号读音正确（包括但不限于：角、垂直于、积分、求和等）。
- [ ] 若使用 EdgeTTS，已告知用户发音限制。

## 讲解与知识点
- [ ] 讲解时明确指向画面中的具体位置。
- [ ] 复杂内容分步呈现，不一次性堆砌。
- [ ] 推导或计算步骤无中间省略。
- [ ] 重要元素有高亮、圈注或箭头指示。
- [ ] 教学路径覆盖七个阶段（激活、直观、定义、运算、反直觉、应用、总结）。
- [ ] 每个反直觉点都有专门的证伪步骤。
- [ ] 每个教学原子独立成步骤，无叠加混乱。

## 字幕验收

- [x] 字幕行数不受限制，超出时自动滚动（由 `subtitle_scroller.py` 处理）
- [ ] 单行字数 ≤ 20 字（统一由 scroller 处理）
- [ ] 字幕滚动速度与语音同步（约4字符/秒）
- [ ] 字幕与语音时长匹配（误差 < 0.2 秒）

## 全场视频专项（如适用）
- [ ] 各场景时长与合并后总时长的误差小于 0.5 秒。
- [ ] 场景切换处无黑屏、卡顿、音画断层。
- [ ] 各场景音量一致（无忽大忽小）。
- [ ] 字幕全场连续，无漏句。
- [ ] 过渡语音与实际下一个场景内容匹配。
- [ ] 各场景分辨率、帧率一致。
- [ ] 片头片尾完整（如有）。

## 教学内容文件对应
- [ ] 视频内容与输出的 JSON 教学内容文件完全一致（所有原子均已实现）。
- [ ] 每个教学原子在视频中有对应的动画步骤。

## 图形内容验收

### 通用图形规范
- [ ] 所有几何图形顶点由数学公式推导，无硬编码坐标
- [ ] **坐标系强制使用 `axes.c2p()` 转换**，禁止手写 `(x*scale, y*scale)` 坐标
- [ ] 线性代数图形展示了基向量或坐标变换
- [ ] 图形动画与 visual_action 配合正确

### 标注规范（LaTeX 优先）
- [ ] 物理量数值与单位之间留有一个空格（`5 m/s` ✅ / `5m/s` ❌）
- [ ] 向量使用粗体或加箭头两种方式之一明确区分（同一场景内保持一致）
- [ ] 下标正确：纯数字下标用 `_1/_2`，中文下标用 `_{\text{共}}` 包裹

### 颜色合规（力学/电磁场景）
- [ ] 力学力-色映射：G=红 / f=黄 / F=绿 / N=蓝 / T=紫（域内 Δ≥51°）
- [ ] 电磁力-色映射：F_E=玫红 / F_A=青绿 / f_L=深紫（Δ≥81°）
- [ ] 浮力颜色：F_浮=青，且与背景有明显对比度

### 物理学科（如适用）
- [ ] 力矢量从**接触点**出发，合力标注 `#9CA3AF` 中灰色
- [ ] 运动轨迹使用虚线 + 箭头（`DashedLine` + `ArrowTip`）
- [ ] 约束物标注正确：绳/杆/弹簧/滑轮各按规范
- [ ] 电路图元符号标准（IEC 60617 / GB/T 4728）
- [ ] 导线 T 型连接处有 `Dot`，L 型连接处无圆点
- [ ] 浮力场景：液体半透明、`V_{\text{排}}` 标注、液体表面水平标线
- [ ] 浮力场景通过 `allowed_overlap_patterns` 豁免物体-液体重叠检测

## 验收结论
- [ ] 所有画面项通过
- [ ] 所有音频项通过
- [ ] 所有讲解项通过
- [ ] 全场专项（如适用）通过
- [ ] 图形内容项通过
