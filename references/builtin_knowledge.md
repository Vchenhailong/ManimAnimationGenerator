## `duration` 字段填写规范（强制）

`duration` **必须**基于 `speech` 字段的字符数计算，**禁止填固定值**或照抄示例的 `6.0`。完整规则见 [`json_schema.md` 第 5.2 节](json_schema.md) 和 [`tts_guide.md` T-D4 节](tts_guide.md)，本节为速查。

**计算公式**：

```
SPEECH_SPEED = 4.0   # 字符/秒（中文朗读速度基准）
min_duration = ceil(len(speech.strip()) / SPEECH_SPEED)
duration    = max(min_duration, 3.0)   # 至少 3 秒
```

**约束**：

- `duration < 3.0` → 自动修正为 3.0（最短朗读时长，见 SKILL.md G-tts-2）
- `duration > 20.0` → **必须拆分原子**（将当前 atom 拆为多个子步骤，每个 duration ≤ 20.0）
- `speech` 为空 → `duration` 取 2.5（无字幕最小值：动画 0.5s + 缓冲 2s）
- 含公式的 `speech` → 需先通过 `math_symbols_to_speech()` 映射符号（见 SKILL.md G-tts-4），再按映射后的中文文本计算字符数

**计算示例**：

| speech（已符号映射）                                                                          | 字符数 | min_duration  | 实际 duration       | 说明          |
| --------------------------------------------------------------------------------------------- | ------ | ------------- | ------------------- | ------------- |
| `"受力分析是力学的基础方法"`                                                                  | 13     | 3.25 → ceil=4 | 4.0                 | 简短陈述      |
| `"牛顿第二定律表明，物体加速度的大小与所受合外力成正比，与质量成反比。"`                      | 32     | 8.0           | 8.0                 | 标准讲解      |

**反例**（禁止）：

- 照抄示例填 6.0：speech 共 45 字却填 duration 6.0（应为 12.0）
- duration 与 speech 长度严重不匹配：speech 仅 1 字却填 10.0（应为 3.0）
- 单原子超过 20s 未拆分

## 电路分析：欧姆定律与串并联（来源：人教版高中物理选修3-1，Halliday《物理学基础》第27章）

前置知识：电流、电压、电阻的基本概念，电荷守恒，能量守恒。
原子序列（8个）：

1. 激活：回顾 I = U / R，电流从正极流向负极
2. 直观体验：展示一个简单回路（电池+灯泡），电流流动动画
3. 串联电路：电流处处相等，总电阻 = R1+R2，电压分配（分压）
4. 并联电路：电压相等，总电流 = I1+I2，总电阻的倒数 = 1/R1+1/R2
5. 混联电路：逐步化简方法（先并后串）
6. 基尔霍夫定律：节点电流代数和为零（KCL）；回路电压代数和为零（KVL）
7. 反直觉点澄清：短路时电阻≈0，电流极大；开路时电流=0，电压可能不为0
8. 应用：家庭电路过载保护原理（并联电器过多，总电流过大）

## 磁场方向判断（安培定则 / 右手螺旋定则）（来源：人教版高中物理选修3-1，Halliday《物理学基础》第29章）

前置知识：磁感线、磁场方向定义（N→S）。
原子序列（5个）：

1. 激活：回顾条形磁铁的磁感线分布（N极出发，S极进入，内部从S到N）
2. 通电直导线的磁场：右手握住直导线，拇指指向电流方向，四指弯曲方向即磁场环绕方向

5. 反直觉点澄清：磁场方向与电流方向是垂直环绕关系，不是同向；螺线管内部磁场从S指向N

## 通电导体在磁场中的受力（左手定则）（来源：人教版高中物理选修3-1，Halliday《物理学基础》第28章）

前置知识：磁场方向、电流方向，安培力公式 F = BIL sinθ。
原子序列（6个）：

1. 激活：回顾磁场对电流的作用力（电动机原理）
2. 左手定则直观演示：磁感线穿手心，四指指电流，拇指指受力方向
3. 垂直情况：B ⊥ I 时受力最大 F = BIL
4. 平行情况：B ∥ I 时受力为 0
5. 反直觉点澄清：力的方向既垂直于磁场，又垂直于电流（三维垂直关系）
6. 应用：直流电动机换向器原理

## 电磁感应（右手定则 / 楞次定律）（来源：人教版高中物理选修3-2，Halliday《物理学基础》第30章）

前置知识：磁通量、法拉第电磁感应定律 ε = -dΦ/dt。
原子序列（7个）：

1. 激活：回顾磁通量变化产生感应电动势
2. 右手定则（切割磁感线）：伸开右手，拇指指向导体运动方向，磁感线穿手心，四指指向感应电流方向
3. 楞次定律直观：感应电流的磁场总是阻碍原磁通量的变化（“增反减同”）
4. 反直觉点澄清：感应电流方向由磁通量变化趋势决定，不是由原磁场单独决定
5. 实例：发电机原理（线圈转动，电流方向交替）
6. 反直觉点澄清：楞次定律中的“阻碍”是“减缓变化”，不是“阻止”
7. 应用：涡流、电磁阻尼、变压器原理简介
