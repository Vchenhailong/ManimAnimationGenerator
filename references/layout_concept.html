Manim 布局概念与预期布局

标准 16:9 配置（frame_height=8, frame_width≈14.22） · 动态计算算法 · ZoneConstants.compute()

1. 动态计算算法（核心）

总窗口 = (config.frame_width, config.frame_height)  → 通过属性获取
安全区 = 总窗口 - 边距比例 × 2  (MARGIN_RATIO = 5%)
标题区高度 = max(安全区高 × 2/10, Text(font_size=40).height)  → 取比例与实测的较大值
字幕区高度 = max(安全区高 × 1/10, Text(font_size=18).height)  → 取比例与实测的较大值
内容区高度 = 安全区高 - 标题高 - 字幕高 - 间距
各栏宽度 = 安全区宽 × 比例（有图形/无图形不同）
调用方式： zones = ZoneConstants.compute(frame_width, frame_height)
cols = ZoneConstants.compute_column_layout(zones, num_columns=2, has_graphics=True)

2. 分栏布局算法（串联流程）

分配宽度 → 内容适配（换行） → 计算高度 → 顶部对齐 → 检测溢出 → 必要时拆分 → 重新分配 → 直到通过
分栏模式有图形时无图形时（均分）
两栏左 60% / 右 40%左 50% / 右 50%
三栏左 30% / 中 30% / 右 40%各 1/3 均分
溢出处理优先级： ① 缩小字号（缩到下限 80%） → ② 换行（align* 环境） → ③ 拆分（split_atom.py 按栏位拆）

3. 安全区域与字幕区高度

标准 16:9 配置：frame_height=8, frame_width≈14.22
安全区：X ∈ [-6.75, 6.75]，Y ∈ [-2.8, 4.0]（高 6.8 单位）
字幕区：Y ∈ [-3.85, -2.8]（高度 ≥ 安全区×10% ≈ 0.68 单位）
标题区：Y ∈ [3.11, 4.0]（高度 ≥ 安全区×20% ≈ 1.36 单位）

实测文字高度（Text().height）vs 比例值 → 取 max()

4. 分栏布局可视化（8 种场景）

场景1 · 单栏 + 主内容
单栏：安全区全部 · 无图形 · 无标题 · 无字幕
场景2 · 单栏 + 标题 + 主内容
2:8 分配 · Y ∈ [3.11, 4.0] 标题 · Y ∈ [-2.5, 3.11] 内容
场景3 · 单栏 + 主内容 + 字幕（9:1）
内容 : 字幕 = 9 : 1 · 字幕 Y ∈ [-3.85, -2.8]
场景4 · 单栏 + 标题 + 主内容 + 字幕（2:7:1）
2:7:1 分配 · 安全区 Y ∈ [-2.8, 4.0] · 字幕 Y ∈ [-3.85, -2.8]
场景5 · 两栏 + 有图形
左栏 60% / 右栏 40% · 左=步骤/公式 · 右=图形
场景6 · 两栏 + 标题 + 有图形
2:7:1 分配 · 两栏 Y ∈ [-2.5, 3.11] · 左 60% / 右 40%
场景7 · 三栏 + 无图形
均分 1/3 · 左=概念/步骤 · 中=公式 · 右=应用/效果
场景8 · 三栏 + 标题 + 有图形（2:7:1）
有图形时：左 30% / 中 30% / 右 40% · Y ∈ [-2.5, 3.11]
